我的技能

HTML5

HTML5是对旧有的 HTML 的补充，取代了 HTML 4.01 和 XHTML 1.0 的标准，新增内容有 video,audio,canvas等。广义来说，HTML5 的概念应该是 HTML，CSS 和 Javacript 一系列技术的整合。

VUE React

Vue，React组建化的开发方式给前端开发带来了便利，可以将一个复杂的ui界面拆分成一个个组件，每一个组件都能根据输入的数据展示出特定的UI效果，最后将一个个组件拼接起来，传递数据流。给团队开发带来很多便利

jQuery

jQuery是一个快速、简洁的JavaScript框架，它封装JavaScript常用的功能代码，提供一种简便的JavaScript设计模式，优化HTML文档操作、事件处理、动画设计和Ajax交互。

PHP

PHP 独特的语法混合了 C、Java、Perl 以及 PHP 自创新的语法。快速的执行动态网页。PHP是将程序嵌入到HTML文档中去执行，执行效率高。PHP支持几乎所有流行的数据库以及操作系统。消耗相当少的系统资源。

nodejs

nodejs作为一个新兴的前端框架,单线程,非阻塞型的事件驱动， 用于方便地搭建响应速度快、易于扩展的网络应用。轻量和高效，非常适合在分布式设备上运行数据密集型的实时应用。

Express

一个简洁而灵活的 node.js Web应用框架, 提供一系列强大特性帮助你创建各种Web应用。Express在nodejs基础上扩展了Web应用所需的功能。丰富的HTTP工具以及来自Connect框架的中间件随取随用，创建强健、友好的API变得快速又简单
